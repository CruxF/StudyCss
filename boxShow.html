box-shadow: offset-x(X轴偏移量) offset-y(Y轴偏移量) blur(阴影模糊半径) spread(阴影扩展) color(阴影颜色) inset(投影方式);

通过设置blur产生阴影效果(水平垂直偏移为0也可以有阴影)：box-shadow: 0 0 20px;

通过设置spread正值产生阴影效果(水平垂直偏移为0也可以有阴影)：box-shadow: 0 0 20px 5px;

通过设置spread负值产生阴影效果(水平垂直偏移为0也可以有阴影)：box-shadow: 0 0 20px -5px;

通过设置水平垂直偏移得到阴影效果：box-shadow: 0 0;

通过设置水平垂直偏移得到阴影效果：box-shadow: 1px 1px;

通过设置水平垂直偏移得到阴影效果：box-shadow: 10px 10px;

通过设置水平垂直偏移得到阴影效果：box-shadow: 0 0 inset;

通过设置水平垂直偏移得到阴影效果：box-shadow: 1px 1px inset;

通过设置水平垂直偏移得到阴影效果：box-shadow: 10px 10px inset;

投影方式：box-shadow: 10px 10px teal;

投影方式：box-shadow: 10px 10px teal inset;

投影方式：box-shadow: 10px 10px teal inset,10px 10px teal;

如果元素同时指定border-radius属性，则阴影呈现相同的圆角：box-shadow: 0 0 10px 10px green;

多重阴影：box-shadow: 0 -5px red,5px 0 orange,0 5px yellow,-5px 0 green;

多重阴影：box-shadow: 0 -5px 5px red,5px 0 5px orange,0 5px 5px yellow,-5px 0 5px green;

多重阴影：box-shadow: 0 -5px 50px red,5px 0 5px orange,0 5px 5px yellow,-5px 0 5px green;

用spread模拟实现border：box-shadow: 0 0 0 1px red;

用spread模拟实现border：box-shadow: 0 0 0 3px red,0 0 0 6px green,0 0 0 9px blue;

用spread实现双色方括号：box-shadow: -30px -30px 0 -25px red,30px 30px 0 -25px green;
